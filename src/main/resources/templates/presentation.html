Exemple d'application java EE Spring Thymleaf Bootstrap

Cette exemple montre l'utilisation de spring MVC, spring security, thymleaf pour la gestion des étudiants d'un établissement scolaire :

  • Saisie des étudiants
  • Affichage des listes
  • Recherche dans la liste étudiant

Les fonctionnalités abordées
Fonctionnalité Outils Utilisés
gestion des habilitations Spring security
Model MVC Spring MVC
Model de données Spring data et Hibernate
Couche présentation Thymleaf et bootstrap
Module de pagination jquery
